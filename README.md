# FinSight 📊

FinSight is a premium, privacy-first personal finance dashboard designed to provide modern financial clarity in real-time[cite: 1]. Built completely client-side, it empowers users to track their financial flow securely without relying on external databases or cloud storage[cite: 1].

## 🚀 Live Capabilities

* **Privacy-First (Zero Database):** All ledger operations run locally in your browser. Your sensitive financial data never leaves your device, utilizing standard `localStorage` caching[cite: 1, 2].
* **Automated Budget Health Score:** Features a live, dynamic analysis of your spending-to-income ratio to visually flag budget sustainability (ranging from Excellent to Critical)[cite: 1, 2].
* **Interactive Data Visualizations:** Integrated with Chart.js to render real-time breakdowns of expense allocations (Doughnut Chart) and flow comparisons (Bar Chart) that update instantly when transactions are modified[cite: 1, 2].
* **Dual-Theme Interface:** Seamlessly transitions between a premium warm slate light mode and a deep navy charcoal dark mode using optimized CSS custom properties[cite: 1, 3].
* **Interactive Hero Showcase:** Includes an interactive, physics-inspired 3D perspective card tilt effect on the landing page that reacts smoothly to user cursor movements[cite: 1, 2].

## 🛠️ Technology Stack

* **Frontend:** Semantic HTML5[cite: 1], Advanced CSS3 (Custom Variables, Flexbox, Grid, CSS Perspective Animations)[cite: 3]
* **Scripting & Logic:** Vanilla JavaScript (ES6+) for DOM manipulation, transactional state tracking, and local storage management[cite: 2]
* **Libraries:** Chart.js (Dynamic Data Visualizations)[cite: 1, 2], Lucide Icons (Vector UI Iconography)[cite: 1, 2]

## 📦 Quick Start & Setup

Because FinSight is entirely localized and lightweight, there are zero dependencies or build pipelines to install.

1. Clone or download this repository.
2. Simply open the `index.html` file in any modern web browser to run the live dashboard instantly[cite: 1, 2].
